Краткая информация

![Image alt](https://github.com/thelightcome/discord-clone/raw/main/screen.png)

- крайние левые кнопки меняют только состояния;
- каретка вниз открывает простой список;
- список чатов. можно выбирать чат;
- логотип в футере навигации открывает информационный блок;
- иконка микрофона и наушника меняют только состояния;
- кнопка настройки открывает новую страницу;
- в поле для ввода текста можно отправляет сообщение только после выбора чата.
